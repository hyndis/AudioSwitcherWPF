# AudioSwitcherWPF

A simple program to let you switch between 2 audio sources, perfect for children to easily switch between headphones and speakers.

Just pick your two sources in settings and save it. Done!

AudioSwitcher beta 2 - Ready to use-file:
https://github.com/hyndis/AudioSwitcherWPF/releases/download/release/AudioSwitcher_b2.rar

Installation:
Unpack the rar-file to a location of your choice and run audioswitcher.exe
